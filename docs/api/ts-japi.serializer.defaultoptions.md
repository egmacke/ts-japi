# Serializer.defaultOptions property

Default options. Can be edited to change default options globally.

**Signature:**

```typescript
class Serializer {static defaultOptions: {
        idKey: string;
        version: string;
        onlyIdentifier: boolean;
        nullData: boolean;
        asIncluded: boolean;
        onlyRelationship: boolean;
        cache: boolean;
        depth: number;
        projection: null;
        linkers: {};
        metaizers: {};
    };}
```
